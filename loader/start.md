# 安装和启动

## 1. 安装Python2.7

由于框架中推荐使用scss，需要依赖Python2.7 的版本，因此在安装工程环境之前，需要先安装好Python2.7，并配置好系统环境变量。

**【注意】要求安装是Python2.7版本，不是Python3.x版本**

安装配置Python教程：  http://blog.csdn.net/u011781521/article/details/53909151

也可以自行网上搜索相关的资料


## 2. 使用npm 官方源安装

```
npm install
```


## 3. 前端开发工程启动

```
npm start
```
