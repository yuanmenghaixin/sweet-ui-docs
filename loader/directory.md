# 目录结构

## 一级目录结构

```
├── .babelrc                // babel配置文件
├── .editorconfig           // 编辑器配置文件
├── .eslintignore           // eslint过滤规则配置文件
├── .eslintrc.js            // eslint校验规则配置文件
├── .gitignore              // git过滤规则配置文件
├── CHANGELOG.md            // 更新日志
├── LICENSE                 // 协议文件
├── README.md
├── build                   // 编译脚本
├── config                  // 环境配置文件目录
├── dist                    // 打包编译输出目录
├── gulpfile.js             // gulp配置文件
├── package-lock.json       // npm 模块安装锁定版本配置文件
├── package.json
├── src                     // 开发资源目录
└── yarn.lock               // yarn 模块安装锁定版本配置文件
```

> 对于框架使用者来说，只需要关注 `config` 和 `src` 这两个文件夹里面的内容
